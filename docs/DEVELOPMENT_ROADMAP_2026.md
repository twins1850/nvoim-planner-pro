**목표**: 각 플래너가 자체 API 키를 사용하도록 시스템 개선

#### 데이터베이스 스키마
```sql
-- planner_api_keys 테이블 생성
CREATE TABLE planner_api_keys (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  planner_id UUID REFERENCES auth.users(id) ON DELETE CASCADE,
  openai_api_key TEXT, -- 암호화 필요
  azure_speech_key TEXT, -- 암호화 필요
  azure_region TEXT DEFAULT 'koreacentral',
  is_verified BOOLEAN DEFAULT false,
  is_active BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW(),
  UNIQUE(planner_id)
);
```
